# Description
基于Unity开发的简单桌面宠物，目前实现功能：
1. 久坐提醒
2. 喝水提醒
3. 拖拽喂食（删除文件）

# Unity Version
Unity 2022.3.34f1c1

# Usage
1. 克隆当前仓库并使用Unity打开
2. build Unity项目
3. **将项目下`\Assets\Config`文件夹移动至build后程序的`\DesktopPet_Data`文件夹下即可运行**
4. 鼠标左键拖动，右键打开设置菜单，拖拽文件至宠物上可以投喂（删除文件）

# Note
- 窗口的穿透点击效果使用win api实现，程序仅能在Windows下正常运行。
- 程序内美术资源均来自Unity Assert Store，使用时请遵守相关规定。
